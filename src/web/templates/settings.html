LLM Provider

What is an LLM? — A large language model is the AI that listens to what you say and generates thoughtful responses to guide your meditation.

Anthropic (Subscription) — Uses your existing Claude Pro or Max subscription via the locally-installed claude CLI (Claude Code). Starting June 15, 2026, you will need to claim a credit to continue using this with no additional cost; Anthropic will email you with instructions.

Ollama (Local) — Free and private. Runs the AI entirely on your computer. We'll recommend the smartest model that your computer can run.

Anthropic / OpenAI / Groq / OpenRouter / Venice (via API Key) — Cloud AI. You sign up with the provider, get an API key, and paste it here. These are pay-per-use; cost depends on the model/provider. Groq also has a generous free tier (rate-limited). Venice has the best privacy.
